======Run app guide=========

1. Extract this rar file to a folder, for example: c:\tmp, so the working folder is c:\tmp\estore.

2. Install node.

3. Open cmd window, run the following command to start the node app:
cd c:\tmp\store
npm install
node ./

4. Open web browser, such as firefox,chrome,IE. Go to the URL of http:\\localhost:8080 to show login page, input "aa" as username and "11" as password to login.
